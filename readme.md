## 🎯 О проекте

**PDF Booklet Creator Pro** — это десктопное приложение с современным веб-интерфейсом, предназначенное для автоматического преобразования обычных PDF-файлов в готовые к печати буклеты. Приложение идеально подходит для подготовки брошюр, каталогов, методических материалов и другой полиграфической продукции.

## ✨ Возможности

### 🎯 Основные функции
- ✅ **Автоматическое создание буклетов** из многостраничных PDF
- ✅ **Визуальный предпросмотр** каждой страницы перед печатью
- ✅ **Галерея страниц** с нумерацией и подсветкой оборотных сторон
- ✅ **Гибкая настройка макета**:
  - 🔄 Поворот всех/каждой второй страницы
  - ↔️ Горизонтальное и вертикальное отражение
  - 📐 Автоподгонка под формат A4 (альбомная ориентация)

### 🛠 Дополнительные инструменты
- 🔍 **Встроенный файловый менеджер** с навигацией по директориям
- 🔎 **Поиск файлов** с фильтрацией по PDF
- 💾 **Сохранение настроек** между сессиями
- 📝 **Система логирования** действий пользователя
- 🎯 **Экспорт в высококачественный PDF** (300 DPI)
- 🚀 **Пакетная обработка** нескольких файлов

🚀 Установка
📋 Требования
🐍 Python 3.8 или выше

🌐 Chrome/Chromium браузер (рекомендуется)

💾 500 МБ свободного места на диске

🖥️ 4 ГБ ОЗУ (рекомендуется

# 1. Клонирование репозитория
git clone https://github.com/yourusername/pdf-booklet-creator.git
cd pdf-booklet-creator

# 2. Создание виртуального окружения (рекомендуется)
python -m venv venv

# 3. Активация виртуального окружения
# 🪟 Для Windows:
venv\Scripts\activate
# 🐧 Для Linux/Mac:
source venv/bin/activate

# 4. Установка зависимостей
pip install -r requirements.txt

# 5. Запуск приложения
python main.py

📦 Пошаговая установка
